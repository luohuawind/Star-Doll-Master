# 星际玩偶师

一个正在开发中的星际题材角色扮演游戏。

## 当前状态

这是一个早期版本，代码有些杂乱，但它能跑起来。  
它包含：
- 完整的角色选择流程（7个好感等级 × 2个种族 × 特殊角色）
- 游戏加载动画和倒计时特效
- 剧情系统的初步框架

## 如何运行

1. 安装 Python 3.8+
2. 安装 pygame：`pip install pygame`
3. 下载代码，运行：`python 星际玩偶师7.0.py`

## 关于这个项目

代码可能不够整洁，但它代表了我从零开始学习游戏开发的过程。  
我会持续更新和完善它。
